이 프로젝트는 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app)으로 부트스트랩된 [Next.js](https://nextjs.org) 기반 UI 애플리케이션입니다.

## 시작하기

아래의 명령어로 시작할 수 있습니다.

```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
# 또는
bun dev
```

[http://localhost:3000](http://localhost:3000)에 접속하면 결과 화면을 확인할 수 있습니다.

초기 화면은 `app/page.tsx`를 수정하면 바로 반영됩니다.

이 프로젝트는 [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)를 사용해 [Geist](https://vercel.com/font) 폰트를 자동으로 최적화하여 로딩합니다.
